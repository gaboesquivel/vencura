# Vecura UI

Frontend application for interacting with the Vencura API - a custodial wallet platform (the Venmo of wallets).

## Features

- Dynamic SDK authentication
- Wallet creation and management
- Balance checking
- Message signing
- Transaction sending
- Basic UI for API interaction

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Dynamic SDK
- Tailwind CSS
- Shadcn/ui components
