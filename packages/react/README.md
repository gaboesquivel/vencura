# @repo/react

React Query hooks generated by hey-api from OpenAPI specs.

## Overview

Provides React Query hooks generated from OpenAPI specs via hey-api.

## Usage

```ts
import { ReactApiProvider } from '@repo/react'
import { useHealthCheck } from '@repo/react'

function App() {
  return (
    <ReactApiProvider config={{ baseUrl: process.env.NEXT_PUBLIC_API_URL! }}>
      <MyComponent />
    </ReactApiProvider>
  )
}

function MyComponent() {
  const { data } = useHealthCheck()
  // data is fully typed from OpenAPI spec
}
```

## Architecture

- ✅ React Query hooks generated by hey-api

- ✅ Type-safe from OpenAPI specs
